# E-Wallet and Bank Logos

This folder contains logos for Philippine e-wallets, banks, and financial services.

## Recommended Logo Files:

### E-Wallets
- `gcash.png` - GCash logo
- `maya.png` - Maya (formerly PayMaya) logo
- `gotyme.png` - GoTyme Bank logo
- `seabank.png` - SeaBank logo
- `paymongo.png` - PayMongo logo
- `coins-ph.png` - Coins.ph logo
- `grabpay.png` - GrabPay logo
- `shopeepay.png` - ShopeePay logo

### Banks
- `bdo.png` - BDO logo
- `bpi.png` - BPI logo
- `metrobank.png` - Metrobank logo
- `unionbank.png` - UnionBank logo
- `landbank.png` - Landbank logo
- `security-bank.png` - Security Bank logo
- `rcbc.png` - RCBC logo

### Others
- `cash.png` - Generic cash icon
- `credit-card.png` - Generic credit card icon
- `bank.png` - Generic bank icon

## Usage:
When creating an account, you can reference these logos by filename.
The app will automatically display the logo if it exists in this folder.

## Image Specifications:
- Format: PNG with transparent background (preferred) or JPG
- Recommended size: 200x200 pixels or 512x512 pixels
- File naming: lowercase with hyphens (e.g., `gcash.png`, `maya.png`)
